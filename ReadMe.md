Mindfulgram is a Google Chrome extension that allows you to use Instagram without certain distractions.

## Features
Ability to hide 6 key distractions:
- Stories
- Posts
- Account Pages
- Search Panel
- Explore Page
- Reels

The popup will save your preferences!
